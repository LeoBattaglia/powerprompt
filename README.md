# PowerPrompt
A simple and lightweight TypeScript-Class to create Terminal-Applications on node.js.

## Installation
Run `npm i powerprompt`

## Import
```
import {PowerPrompt} from "powerprompt";
const pp = new PowerPrompt();
--> Example: pp.printLine();
```

## Public Methods
- close
- choose
- input
- multiSelect
- print
- printCommand
- printError
- printInput
- printLine
- printOption
- printSelected
- printTitle
- select

## Parameters
None
